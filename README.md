# Homebrew Tap

Custom Homebrew tap for Scribe and other tools.

## Installation

```bash
brew tap rohenaz/tap
brew install --cask scribe
```

Or in one command:

```bash
brew install --cask rohenaz/tap/scribe
```

## Available Casks

| Cask | Description |
|------|-------------|
| scribe | Timeline-based work evidence aggregator |

## Updating

```bash
brew update
brew upgrade --cask scribe
```
